# Emoji Picker

> 🎣👌 light emoji picker made in React.

## 📄 Getting Started

Run app and page examples.

```sh
$ yarn # install node packages.
$ yarn start # start it.
$ yarn test # test every component with coverage result as well.
```

Run components using styleguide.

```sh
$ yarn start:styleguide
```
